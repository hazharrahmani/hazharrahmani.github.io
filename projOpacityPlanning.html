Opacity-Enforcing Planning


a) An autonomous robot in an adversarial environment. b) Specification DFA for a temporal goal.

Description

The aim of enforcing opacity is to confuse an intruder who even knows the system model and perceives the sequence of observations the system emits, about whether the system performed a secret behavior or entered a set of secret states. This project aims to provide algorithms and theories for opacity-enforcing planning in stochastic environment with complex mission objectives.
